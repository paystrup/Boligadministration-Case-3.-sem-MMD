# Boligadministration Case 3. sem MMD
 
## Case fra 3. sem Frontend Developer multimediedesign på EAA
### En boligforening råder over et stort boligkompleks med lejeboliger. De ønsker sig en digital løsning som giver dem en oversigt over boligernes lejere, lejerens navn, lejlighedsnummer, indflytningsdato, betalt indskud i kroner og månedlig husleje i kroner ekskl. Forbrug!
